Coding - Github Link

  • Personal Website

    It was really fun making this website for myself, to consider it from a UI/design standpoint and create something that represents myself in a creative way. In the process, I practiced my coding skills with HTML/CSS and JQuery.

    Github

  • Card-Guessing iOS game

    Objective-C practice with some terribly drawn graphics. A card is randomly selected, player tries to guess the card through a ScrollView. High score (the lowest number of guesses to get it right) is recorded, and there's an option to give up.

    Github

  • Store Inventory iOS App

    More Objective-C practice--users can add items to their store to track store stock.

    Github

  • Parallelized Prime Number Generation with OpenMP

    This program written in C uses OpenMP to parallelize the generation prime numbers.

    Github

  • Interactive Car Animation made with Processing

    Python and object-oriented programming practice by way of Processing. "T" and "m" keys add trees and mountains to the animation, respectively.

    Github

  • ASCII Art Logger

    Practicing serverside Javascript. Simple journal/blog-like web application that allows user to log entries of ASCII art with tags that allow easy searching.

    Github

  • Link Aggregation Website

    Skeletal Reddit-like link aggregation website with very basic user authentication (Bcrypt password hashing) and user storage with MongoDB. Uses Handlebars for templating and Express for route handling.

    Github

  • Emoji Moodboard

    Practice with DOM manipulation using vanilla Javascript. Web application consists of a 8x8 board of randomly generated emojis users can pin emojis from. The rest of the board can continue to be randomized. Application also displays Simpson's Diversity Index to measure diversity in the board.

    Link
